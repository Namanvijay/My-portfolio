  • Data Verification in Decentralized Systems

    The project aims for providing data verification in decentralized organization where files file is divided into chunks and distributed to peers. Merkle trees are used to store the files as leaves consisting of hashing of small files chunks and root hash stored inauthorized servers. This used SHA-256 for encrypting files with random yet consistent values.
